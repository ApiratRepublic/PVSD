## จุดประสงค์
สำหรับเช็คว่า GDB มี Featureclass ที่กำหนด ครบหรือไม่ และนับจำนวนฟีเจอร์คลาสที่ตรงกับ รูปแบบที่กำหนดไว้ (GDB Data Standard.md) 
สคริปต์นี้สร้างขึ้นเพื่อช่วยตรวจสอบความสมบูรณ์ของข้อมูล GDB 
ที่อาจจะเกิดจากการคัดลอกข้อมูลที่ไม่สมบูรณ์
(เวลาสำเนา gdb ผ่านเน็ตเวิร์กอาจมีปัญหา )

## สิ่งที่ต้องมี
ไพธอนสคริปต์นี้ ผู้ใช้ต้องติดตั้ง arcpy และ pandas ก่อนใช้งานสคริปต์นี้
สามารถรันได้ใน Python environment ที่มี arcpy (เช่น ArcGIS Pro)  

## การตั้งค่า

gdb ทั้งหมด เก็บไว้ใน root directory เดียวกัน ต้องกำหนดในตอนต้นของไฟล์
	- root_dir = ที่รวมไฟล์ GDB
	- output_excel = พาร์ธเก็บเอ็กเซล
ปรับแต่ง root_dir และ output_excel ตามต้องการ

ปรับแต่ง รูปแบบฟีเจอร์คลาสตามต้องการในส่วน # === กำหนดฟีเจอร์คลาสที่จะตรวจสอบ === #
