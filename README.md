# PVSD
PVSD Repository is GIS Data Reviewer.

Specifically for my friends at the Psychosis Veiled in Sacred Dreams.
รวบรวมไพธอนสคริปต์ใช้งานด้าน gis ของเหล่า PVSD

โปรดอ่าน wiki ซึ่งจะอธิบายการใช้ไพธอนสคริปต์เหล่านี้

## Features
### gdb_data_reviewer.py
สำหรับการใช้ตรวจสอบรูปแบบข้อมูลที่อยู่ใน GDB 
[อ่านรายละเอียดที่นี่](https://github.com/ApiratRepublic/PVSD/wiki/gdb_data_reviewer)

### check_required_featureclass.py
สำหรับเช็คว่ามีฟีเจอร์คลาสครบหรือไม่ 
(ไม่จำเป็นต้องใช้ เขียนขึ้นเพราะเจอปัญหาว่า gdb บางแห่งดาวน์โหลดมาไม่สมบูรณ์ เลขตรวจสอบเฉย ๆ)


## Support
หากมีคำแนะนำว่าควรตรวจสอบอะไรเพิ่ม หรือตรรกะในการตรวจสอบผิดพลาด กรุณาแจ้งโดยตรงนะจ๊ะ
